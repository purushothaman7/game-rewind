Returns & Refunds

Last updated: January 2025

Return Policy

We want you to be completely satisfied with your purchase. Our return policy is designed to be fair while protecting the quality of our pre-owned games.

Return Eligibility

Games may be returned within 7 days of purchase if:

  • The game disc is defective or does not work properly
  • The game was significantly misrepresented in condition
  • You have the original receipt
  • The game is in the same condition as when purchased

Non-Returnable Items

The following items cannot be returned:

  • Games without original receipt
  • Games damaged by customer misuse
  • Games returned after 7 days
  • Games with missing or damaged cases (unless purchased that way)

Refund Process

  1. Bring the game and receipt to our store within 7 days
  2. Our staff will inspect the game condition
  3. If approved, refund will be processed immediately
  4. Refunds are provided in the original payment method

Exchange Policy

  • Games can be exchanged for store credit within 14 days
  • Store credit can be used for any future purchases
  • Exchange value is based on current market pricing

Defective Games

If a game doesn't work properly:

  • Return within 7 days for full refund
  • We will test the game to confirm the defect
  • No questions asked if the game is genuinely defective

Store Credit

  • Store credit never expires
  • Can be used for any games in our inventory
  • Transferable to friends and family
